Visual Studio Code

El editor más potente y personalizable para desarrollo web. Extensiones, depuración integrada y terminal para tu flujo de trabajo.

HTML5 + Canvas

Dibuja gráficos en tiempo real, controla sprites y crea animaciones suaves con la potencia del elemento canvas.

Lógica de juego, detección de colisiones, físicas simples y control de eventos con el lenguaje de la web.

Extensiones clave

Live Server, Prettier, ESLint, Debugger for Chrome, y packs para game dev como Phaser o PixiJS.

🔧 Configuración rápida en VS Code

1. Instala Visual Studio Code desde code.visualstudio.com
2. Abre la terminal (Ctrl + ñ) y crea una carpeta para tu juego.
3. Crea un archivo index.html y otro script.js.
4. Instala la extensión "Live Server" para ver los cambios en tiempo real.
5. ¡Empieza a codificar tu juego!
